## About
This repository hosts the project website for our research on safety challenges in agentic fine-tuning of Large Language Models and our proposed solution: Prefix INjection Guard (PING).

## Deployment

This website is hosted on GitHub Pages. To deploy:

1. Push your changes to the `main` branch
2. Enable GitHub Pages in repository settings
3. Set the source to the root directory

The website will be available at `https://agentic-ft-safety.github.io`

The `/paper` route will redirect to the arXiv paper.
